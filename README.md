# GitHub Pages Only Product Photo Tool

Upload these files directly in repository root:
- index.html
- style.css
- script.js
- README.md

Then:
Settings → Pages → Deploy from branch → main → /root

Features:
- Browser AI background removal
- White background fixed
- Product move/rotate/flip/zoom
- Brush + Box Select restore/erase
- Enhance: brightness, contrast, saturation, sharpen, shadow
- 400×400 JPG
- Target 190–199KB and under 200KB

Important:
First use may take time because AI model downloads in browser. Use Chrome or Edge.
